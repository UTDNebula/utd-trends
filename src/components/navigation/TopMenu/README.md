### Props Table

| Prop        | Type      | Description                                                               | Required |
| :---------- | :-------- | :------------------------------------------------------------------------ | -------- |
| `isPlanner` | `boolean` | A boolean that indicates whether the page being shown is the Planner page | Yes      |
